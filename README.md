# ROFL App

The official dashboard for exploring, creating, deploying and monitoring
ROFL-powered applications.

The production deployment lives at <https://rofl.app/>.

> [!TIP]
> If you want to try out the next release on staging, go to
> <https://stg.rofl.app/>.
> If you want to experiment with the latest code in the `master` branch,
> go to <https://dev.rofl.app/>.

## Features

- Create, register and deploy ROFL-powered apps.
- Provision, monitor and manage ROFL machines.
- Modify metadata, secrets and compose files of your ROFL-powered apps.
- View logs and statuses of your ROFL machines.
- Works with multisig accounts such as [Oasis Safe].

[Oasis Safe]: https://safe.oasis.io/

## About ROFL

[Oasis ROFL] (Runtime Offchain Logic) enables building secure
applications running in a trusted execution environment (TEE).

This is ideal for building decentralized trustless oracles, AI agents,
verifiable compute tasks such as AI training, or game servers.

[Oasis ROFL]: https://docs.oasis.io/build/rofl/
